# Word 输出约定

Claude 生成论文正文内容时，**必须**遵守以下格式。

## 输出格式

每段正文输出都以位置标注开头：

```
[章节: 3.2 XBP1 敲低后转录组重塑]
[位置: 结果第 2 段，PCA 与样本相关性]

<正文内容，可直接复制粘贴到 Word>

---
[图注: 图 3.2-2 火山图]
<图注内容>
```

## 规则

1. **纯文本**——不使用 Markdown 格式（不用 `**加粗**`、`# 标题`）。用户在 Word 中自行排版。
2. **一次一节**——不要一次输出多个章节。
3. **保留所有数字**——不得四舍五入、改写或省略分析产生的统计数值。
4. **学术中文语体**——正式、第三人称、被动语态。无口语化表达。
5. **基因/蛋白标注**——基因名用下划线包裹（用户在 Word 中改为斜体）：`_XBP1_`；蛋白写"XBP1 蛋白"。
6. **图表交叉引用**——使用格式："如图 3.1-1 所示" / "见表 3.2-1"。
7. **引用占位符**——行内使用 `[AuthorYear]` 格式，如 `[Yoshida2001]`。用户在 Word 中插入引文管理器引用。
8. **双语图注**——被要求时同时提供中文和英文图注。

## 前 3 个 Session 的同步约定

前 3 次会话中，每完成一个关键阶段后附加同步摘要：

```
[同步摘要]
- 完成: 3.1 单细胞结果段落（UMAP + 标记基因 + 细胞比例）
- 改动文件: 无（纯文本输出，用户自行粘贴）
- 待定: 图 3.1-1 图注需要补充 n 和参数
- 下一步: 写 3.1 CellChat 互作段落
- 需要确认: 无
```
